# HKSchoolPlace 枚举与常量定义

本文档统一定义系统中使用的所有枚举值和常量，供前后端、数据库、抓取脚本共同遵守。

---

## 1. 学段 (stage)

| 值 | 中文 | 英文 | 说明 |
|----|------|------|------|
| `kg` | 幼稚園 | Kindergarten | MVP 第一版主要数据 |
| `primary` | 小學 | Primary School | MVP 保留，数据后补 |
| `secondary` | 中學 | Secondary School | MVP 保留，数据后补 |

## 3. Vacancy 状态 (vacancy_status)

| 值 | 中文显示 | 颜色 | 说明 |
|----|----------|------|------|
| `available` | 有學位 | 绿色系 `bg:#ECFDF5 text:#16A34A` | 明确有空位 |
| `waiting` | 候補中 | 黄色系 `bg:#FFFBEB text:#D97706` | 候补 / waiting list / 不保证 |
| `full` | 已滿額 | 灰色系 `bg:#F3F4F6 text:#6B7280` | 明确满额 |
| `unknown` | 未知 | 浅灰色 `bg:#F9FAFB text:#9CA3AF` | 源数据缺失或无法判断 |

## 9. 数据源标识 (source)

| 值 | 说明 | 用途 |
|----|------|------|
| `edb` | 教育局官方 | vacancy + 学校主数据 |
| `data_gov_hk` | 香港政府公开数据 | 学校主数据 |
| `schooland` | Schooland 网站 | 第三方补充 |
| `kgp` | 幼稚园概览网站 | 第三方补充 |
| `manual` | 人工录入 | 后台手动维护 |

---

## 10. 同步状态 (sync_status)

| 值 | 说明 |
|----|------|
| `running` | 同步进行中 |
| `success` | 全部成功 |
| `partial_success` | 部分成功（有解析/匹配失败记录） |
| `fail` | 全部失败 |

---

## 11. 用户角色 (user_role)

| 值 | 说明 |
|----|------|
| `user` | 普通用户（默认） |
| `admin` | 管理员 |

---

## 12. 排序方式 (sort)

| 值 | 中文显示 | 说明 |
|----|----------|------|
| `recommended` | 推薦排序 | 默认。有位优先 → 更新时间倒序 → 名称正序 |
| `updated_desc` | 最近更新 | 按 vacancy 更新时间倒序 |
| `name_asc` | 名稱 A-Z | 按学校名称正序 |

---

## 13. 常量

| 常量 | 值 | 说明 |
|------|----|----- |
| `MAX_COMPARISON` | 5 | 每个用户最多对比学校数 |
| `DEFAULT_PAGE_SIZE` | 20 | 列表页默认每页条数 |
| `MAX_PAGE_SIZE` | 50 | 列表页最大每页条数 |
| `STALE_THRESHOLD_HOURS` | 72 | vacancy 超过此小时数未更新则标记 stale |
| `VACANCY_SYNC_FREQUENCY` | 每日 1-2 次 | vacancy 数据抓取频率 |
| `SCHOOL_SYNC_FREQUENCY` | 每周 1 次 | 学校主数据抓取频率 |
