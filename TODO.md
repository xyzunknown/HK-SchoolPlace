## P0

### O1. 学校管理可编辑化

- 状态：`DONE`
- 目标：让后台学校管理从“只读列表”变成“可维护主数据”
- 完成内容：
  - 新增 `/admin/schools/[id]` 学校编辑页
  - 支持编辑学校名称、地址、电话、网站、类型、时段、学费、计划参与、展示状态等字段
  - 学校列表页新增筛选、编辑入口、前台查看入口
- 验收结果：
  - 管理员可从 `/admin/schools` 进入编辑页
  - 保存后可通过现有 `PATCH /api/admin/v1/schools/:id` 落库
  - 已验证：`npm run typecheck`、`npm run build`

### O2. Vacancy 管理与同步日志排查增强

- 状态：`DONE`
- 目标：让后台能直接处理 stale vacancy 与同步失败排查
- 完成内容：
  - 新增 `/admin/vacancies`
  - 支持按学校、状态、stale 条件筛选 vacancy
  - 支持编辑 `is_stale` 与 `admin_note`
  - `/admin/sync-logs` 增加状态/来源筛选与失败排查快捷入口
- 验收结果：
  - 管理员可直接查看和处理 vacancy 记录
  - 日志页可快速跳去 unmatched 和 stale vacancy 处理页
  - 已验证：`npm run typecheck`、`npm run build`
